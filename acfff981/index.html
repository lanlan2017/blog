批处理set命令详解

set命令用于在当前cmd窗口中设置环境变量，这些环境变量只对当前cmd窗口有效。
set /a可以进行数学运算，如执行加减乘除，求余等。
set /p可以读取一行键盘输入。

set帮助文档

dos计算精确范围

DOS计算只能精确到32位，这个32位是指二进制32位，其中最高位为符号位（0为正，1为负），低位31位为数值。 31个1换成十进制为2147483647，所以DOS计算的有效值范围是-2147483648至2147483647，超出该数值范围时计算出错
