shell内部开关

  • set -u：当引用一个未定义的变量时，产生一个错误
  • set +u：当引用一个未定义的变量时，认为是一个空串（默认情形）
  • set -x：执行命令前打印出shell替换后的命令及参数，为区别于正常的shell输出，前面冠以+号
  • set +x：取消上述设置

命令echo

语法与功能

打印各命令行参数，每两个参数之间用一空格分开，最后打印换行符
