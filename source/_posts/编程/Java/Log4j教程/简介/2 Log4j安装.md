# 使用下载好的lib
下载`lib`,然后添加到`build path`.
# Maven依赖
到项目文件夹并找到`pom.xml`，添加以下依赖关系:
```xml pom.xml
<dependency>
    <groupId>log4j</groupId>
    <artifactId>log4j</artifactId>
    <version>1.2.17</version>
</dependency>
```
