## 2.5小结 ##
`URL`重写和隐藏域是轻量级的会话跟踪技术，适用于那些仅跨少量页面的数据。而`cookies`和`HttpSession`对象，更加灵活但也有限制，尤其是在应用`HttpSession`时会消耗服务器内存。
