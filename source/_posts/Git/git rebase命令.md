# git rebase的作用
可以对某一段线性提交历史进行编辑、删除、复制、粘贴；因此，合理使用rebase命令可以使我们的提交历史干净、简洁
# 前提
不要通过rebase对任何已经提交到公共仓库中的commit进行修改。
