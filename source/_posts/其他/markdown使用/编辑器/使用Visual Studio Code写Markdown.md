# 快捷键支持 #
安装`Markdown All in One`插件,该插件提供了快捷键支持.
## 加粗 ##
Ctrl+B
## 切换代办列表 ##
Alt+C
## 格式化表格 ##
Alt+Shitf+F

# 快捷键表格 #

| `快捷键`|`功能`|
|:--|:--|
| `Ctrl + B`|`切换粗体`|
| `Ctrl + I`|`切换斜体`|
| `Ctrl + Shift + ]`|`提升标题`|
| `Ctrl + Shift + [`|`降低标题`|
| `Ctrl + M`|`切换公式`|
| `Alt + C`|`勾选或者取消勾选任务列表` |
| `Ctrl + Shift + V`|`切换预览`|
| `Ctrl + K V`|`将预览切换到侧边`|

## 小结 ##
快捷键支持并不完全.还是算了.
