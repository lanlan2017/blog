Eclipse 终端Terminal汉字重复问题

发表于 2021-09-01 更新于 2022-04-04 分类于 Java ， Java EE ， 环境搭建 Waline： 本文字数： 175 阅读时长 ≈ 1 分钟

问题描述

Eclipse如何打开终端

汉字重复问题

解决方案

使用旧版的cmd控制台

Win+R启动运行，然后输入cmd

在cmd窗体顶部按下右键，选择属性

在选项中勾选使用旧版控制台，然后关掉cmd

在Eclipse中重新打开终端即可。
